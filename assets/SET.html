SET คืออะไร?

ตลาดหลักทรัพย์แห่งประเทศไทย(The Stock Exchange of Thailand,SET) เป็นนิติบุคคลที่จัดตั้งขึ้นโดยพระราชบัญญัติตลาดหลักทรัพย์แห่งประเทศไทย พ.ศ. 2517 อยู่ภายใต้การกำกับดูแลโดย สำนักงานคณะกรรมการกำกับหลักทรัพย์และตลาดหลักทรัพย์ (ก.ล.ต.) เปิดทำการซื้อขายขึ้นอย่างเป็นทางการครั้งแรกใน วันที่ 30 เมษายน พ.ศ. 2518 ทำหน้าที่เป็นตลาดรองเพื่อแลกเปลี่ยนซื้อขายตราสารทุน ของบริษัทต่าง ๆ ที่จดทะเบียนไว้ และเพื่อให้สามารถระดมเงินทุนเพิ่มเติมจากสาธารณะ
ได้โดยสะดวก ปัจจุบันการดำเนินงานของตลาดหลักทรัพย์แห่งประเทศไทย อยู่ภายใต้พระราชบัญญัติหลักทรัพย์และตลาดหลักทรัพย์ พ.ศ. 2535

บทบาทและหน้าที่ของ SET

การซื้อขายหลักทรัพย์ : SET เป็นสถานที่ที่นักลงทุนสามารถซื้อขายหุ้น ตราสารหนี้ กองทุนรวม และหลักทรัพย์อื่น ๆ ได้อย่างสะดวกและปลอดภัย
การระดมทุน : SET เป็นช่องทางที่บริษัทสามารถระดมทุนจากประชาชนทั่วไปได้ โดยการเสนอขายหุ้นหรือหลักทรัพย์อื่น ๆ ผ่านการจดทะเบียนในตลาดหลักทรัพย์
การส่งเสริมการลงทุน : SET มีบทบาทในการส่งเสริมความรู้และความเข้าใจเกี่ยวกับการลงทุนให้กับประชาชนทั่วไป เพื่อเพิ่มจำนวนและคุณภาพของนักลงทุนในตลาดหลักทรัพย์
การกำกับดูแลและควบคุม : SET มีการกำกับดูแลและควบคุมการทำธุรกรรมในตลาดหลักทรัพย์ เพื่อให้เป็นไปตามกฎหมายและระเบียบที่เกี่ยวข้อง เพื่อรักษาความเป็นธรรมและความโปร่งใสในตลาด

ดัชนีสำคัญของ SET

SET Index: ดัชนีหลักของตลาดหลักทรัพย์แห่งประเทศไทย ซึ่งสะท้อนภาพรวมของการเคลื่อนไหวของราคาหุ้นทั้งหมดที่จดทะเบียนในตลาดหลักทรัพย์
SET50 Index: ดัชนีที่รวบรวมหุ้น 50 ตัวที่มีมูลค่าตลาดและสภาพคล่องสูงที่สุดในตลาดหลักทรัพย์แห่งประเทศไทย
SET100 Index: ดัชนีที่รวบรวมหุ้น 100 ตัวที่มีมูลค่าตลาดและสภาพคล่องสูงที่สุดในตลาดหลักทรัพย์แห่งประเทศไทย
